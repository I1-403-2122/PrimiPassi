Capire HelloWorld

Ora che hai eseguito il programma HelloWorld, vediamo come funziona.

Ecco nuovamente il codice sorgente:

/**
 * La classe HelloWorld è un programma che
 * stampa la stringa “HelloWorld!!” sullo standard output.
 */

public class HelloWorld {

// Display "Hello World!"
    public static void main(String[] args) {
        System.out.println("HelloWorld!!");
    }
}

Il programma HelloWorld si compone di tre parti principali:

  • I commenti

  • La definizione della classe HelloWorld

  • Il metodo main

I commenti

I commenti sono ignorati dal compilatore ma sono utilizzati dai programmatori per lasciare delle indicazioni nel codice sorgente. Nel codice sorgente di HelloWorld troviamo due tipi differenti di commenti:

Commenti su una o più righe

Il commento inizia con /* e termina con */.

Esempi:

/* mio commento. */


/* 
    il mio commento
    su tre
    righe
*/

Commenti su una sola riga

Il commento inizia con // e termina alla fine della riga. Esempi:

// il mio commento.

System.out.println("HelloWorld!!"); // un altro commento

La definizione della classe

La definizione della classe HelloWorld è la seguente:

public class HelloWorld {
    //... corpo della classe
}

Secondo quanto mostrato nel riquadro precedente, il modo più semplice di definire una classe in Java è:

class <NomeClasse>{
…
}

La parola chiave class viene posta all’inizio della definizione della classe nominata NomeClasse, e il codice della classe viene scritto nel blocco definito dalle parentesi di apertura { e la parentesi di chiusura }.

Ad esempio per definire una classe che si chiami Prova, scriveremo:

class Prova{
    // corpo della classe
}

Il metodo main

Ecco la dichiarazione del metodo main:


public static void main(String[] args) {
    // corpo del metodo  
}

Nel linguaggio di programmazione Java ogni programma deve contenere un metodo main. La dichiarazione del metodo main è appunto:

public static void main(String[] args) {…}

Le parole risevate public e static possono essere scritte come public static oppure nell’ordine inverso static public, ma è convenzione utilizzare la forma public static come mostrato nell’esempio.

Si può dare all’argomento qualsiasi nome, ma generalmente i programmatori scelgono il nome args o argv.

Tutti gli altri elementi della dichiarazione devono essere scritti esattamente come mostrato nell’esempio; quando scrivi un programma ricordati che Java è case sensitive ossia è sensibile al caso dei caratteri (‘A’ è diverso da ‘a’).

Il metodo main è simile alla funzione main dei linguaggi C e C++ ed è il punto d’inizio di ogni programma Java; è infatti il main che invoca la sequenza di istruzioni che costituiscono il programma.

Il metodo main accetta un singolo argomento: un array di elementi di tipo String.

public static void main(String[] args) {

Questo array è il meccanismo attraverso il quale il sistema operativo passa delle informazioni al programma. Ogni stringa nell’array è chiamata argomento da linea di comando.

Gli argomenti da linea di comando permettono all’utente di modificare il comportamento dei programmi senza doverli ricompilare. Ad esempio, un programma di ordinamento potrebbe permettere all’utente di specificare in quale ordine gli elementi debbano essere disposti tramite un argomento da linea di comando come –crescente oppure -decrescente.

Il programma HelloWorld ignora gli argomenti da linea di comando, ma generalmente bisogna verificare se tali argomenti esistono.

Il corpo del metodo main è definito dalla parentesi di apertura { e la parentesi di chiusura }. Nel corpo del metodo main della classe HelloWorld troviamo una sola istruzione:

System.out.println("HelloWorld!!");

Tale istruzione utilizza la classe System della libreria di base di Java per stampare sullo schermo la scritta “HelloWorld!!”.
