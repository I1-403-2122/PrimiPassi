Installazione

Trovate nano nellâ€™archivio compresso che avete ricevuto insieme a questo documento (nano-2.5.0.zip). Per installare nano su Windows dovete:

  1. Estrarre la cartella (es. nano-2.5.0-win32)

  2. Copiare la cartella estratta nella cartella dei programmi (es. C:\Program Files (x86))

  3. Aggiungere il percorso della cartella di nano (es. C:\Program Files (x86)\nano-2.5.0-win32) alla variabile PATH

Se non funziona, controllare di aver settato correttamente la variabile PATH e riprovare (ricordandosi di chiudere e riaprire la shell dopo ogni modifica della PATH).

Syntax Highlighting

Per attivare la funzione syntax highlighting per Java:

  1. Creare un file java.nanorc in una cartella nota (es. C:\Users\IEUser) con il contenuto seguente:

    syntax "java" "\.java$"
color green "\<(boolean|byte|char|double|float|int|long|new|short|this|transient|void)\>"
color red "\<(break|case|catch|continue|default|do|else|finally|for|if|return|switch|throw|try|while)\>"

color red ""[^"]*""
color yellow "\<(true|false|null)\>"

# Comments
color blue "//.*"
color blue start="/\*" end="\*/"
color brightblue start="/\*\*" end="\*/"

# Trailing whitespace
color ,green "[[:space:]]+$"

  2. Attivare syntax highlighting per Java, nel file di configurazione nano.rc:
