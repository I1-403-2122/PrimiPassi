Scrivere un programma in Java

Apri un a shell (powershell/terminale):

Tutti i comandi utilizzati in questo tutorial funzionano nel Terminale di Mac OS X, nella PowerShell di Windows e pure nelle shell delle distro Linux. Eccone alcuni molto utili che userete spesso e che dovreste imparare a conoscere:

  • ls (list): stampa il contenuto (files e cartelle) della cartella corrente (detta cartella di lavoro).
  • cd (change directory): cambia cartella di lavoro.
  • mkdir (make directory): crea una nuova cartella.
  • rm (remove): cancella un elemento (file o cartella). Con questo comando siate prudenti.
  • pwd (print working directory) : stampa il percorso della cartella di lavoro corrente.
  • man (manual): mostra il manuale di un comando specificato.

I seguenti comandi sono comandi esterni, ossia non fanno parte del sistema operativo, per poterli utilizzare dovete prima installare la JDK e la JRE:

  • javac (java compiler): È il compilatore di Java, fa parte della JDK e serve per trasformare in eseguibili (.class) i programmi scritti in Java (.java).
  • java: È l’interprete Java, fa parte della JRE e serve per eseguire i programmi Java compilati.

Crea una nuova cartella in cui salveremo gli esercizi:

> mkdir Java

Ora spostati nella cartella appena creata:

> cd Java

…e crea un nuovo file sorgente in cui scriverai il tuo primo programma. Usa nano per creare ed editare il file HelloWorld.java:

> nano HelloWorld.java

Attenzione, quando usi javac per compilare un programma:

  • il nome del file deve essere scritto in modo esatto rispettando anche Maiuscole e minuscule.
  • l’estensione deve essere .java.

Ora trascrivi il codice sorgente della classe HelloWorld nel file appena creato:


/**
* La classe HelloWorld è un programma che
* stampa la stringa “HelloWorld!!” sullo standard output.
*/
public class HelloWorld {
  // Display "Hello World!!"
  public static void main(String[] args) {
    System.out.println("Hello World!!");
  }
}

Salva il file utilizzando la combinazione di tasti ^O e poi esci dal programma digitando ^X.

Ora nella cartella di lavoro (Java) dovrebbe esserci un nuovo file che si chiama HelloWorld.java, verificalo digitando il comando ls:


Adesso devi compilare il file HelloWorld.java che hai appena salvato. Per fare questo devi digitare il comando:

> javac HelloWorld.java

Se il file sorgente contiene errori il compilatore ti segnala dei messaggi di errore; in questo caso devi controllare di aver copiato esattamente il codice sorgente nel file HelloWorld.java.

Se tutto è andato bene, la compilazione termina senza avvisi.

Ora nella cartella di lavoro ci deve essere anche il file HelloWorld.class prodotto dal compilatore. Prova ad elencare il contenuto della cartella utilizzando il comando ls:


Non ti resta che eseguire il file HelloWorld.class digitando il comando java HelloWorld:


Attenzione, quando usi il comando java per eseguire una classe:

  • il nome della classe deve essere scritto in modo esatto rispettando anche il caso dei caratteri (maiuscole e minuscole).
  • l’estensione (.class) deve essere omessa.

Se sullo schermo appare la scritta Hello World!! il programma funziona.
